# WorldGuard

Auf dem Server übernimmt [WorldGuard](https://modrinth.com/plugin/worldguard) das managen und beschützen von Regionen.

Das erstellen von Regionen und das Managen von Members sollte generell über den Discord Bot Plotty geschehen. [Hier](/plotty.md) findest du eine Anleitung, wie man den Bot benutzt.

## Commands

### `/rg info`

Gibt eine interaktive Chat-Nachricht mit Informationen über das Grundstück, in dem man sich gerade befindet. Mit einem Klick auf `[Flags]` kann man verschiedene Flags auf seinem eigenen Grundstück setzen, wie z.B. `creeper-explosion`, welches das Explodieren von Creepern verhindert.
